# Real Exam Data

This directory stores metadata and public reference sources for real TP.HCM Grade 10 entrance exams.

## Current Scope

- Exam years currently tracked: 2026, 2025, 2024, 2023, 2022.
- Each year has 3 subjects: Toan, NguVan, TiengAnh.
- This data is kept separate from `data/questions`, which contains AI-generated seed question-bank data.

## Content Status

Exam files currently use `contentStatus: source_link_only`: public sources are recorded for verification, but the full exam text has not been transcribed from embedded images/PDFs unless a reliable permitted text version is available. When an official PDF or permitted text source is available, update the relevant file and change `contentStatus` to `transcribed_verified`.

## Source Types

- `official_mirror_public`: public article or portal that directly references a publication from the Department of Education or a state/public body.
- `mirror_public`: public press or education source; verify against the official Department publication before commercial use.

This data is for reference and study use. Verify against the official TP.HCM Department of Education publication before commercial release.
